# Starlight Kids Theater Co.

This project is a fictional website created for a coding class assignment to demonstrate HTML and CSS skills.

## Site Overview

Starlight Kids Theater Co. is a youth-focused community theater organization that offers acting, musical theater, and backstage tech programs for kids and teens.

The site includes:

- 6 HTML pages (`index`, `about`, `programs`, `events`, `get-involved`, `contact`)
- Shared navigation and footer on every page
- External stylesheet (`styles.css`) with a Google Web Font and hover effects
- Stock imagery (not included in this repo) to represent youth theater activities

## Requirements Covered

- 6 page navigation
- External stylesheet shared by all pages
- At least 10 CSS rules
- Google Web Font (Poppins)
- Pseudo-class `:hover` on navigation and buttons
- Ordered/unordered list on the Programs page
- At least 4 images in the content area
- Email hyperlink (`mailto:`)
- External link opening in a new window
- Footer with text navigation, copyright symbol, year, and “Designed by”
- No lorem ipsum; all copy is readable and portfolio-friendly
